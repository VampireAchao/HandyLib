5. 存储初始化 StorageApi.enableSql();
6. gui事件初始化 HandyClickFactory.init();
7. 统计初始化 MetricsUtil.addMetrics()
8. 连接池初始化 SqlManagerUtil.enableTable()
9. gui按钮初始化 MaterialApi.loadInventoryWriteParam()
